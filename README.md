# Repara i Roda 360 — Optativa de 4t d'ESO · Curs 2026-2027

Taller de reparació de bicicletes combinat amb aula maker per a un grup professionalitzador de 4t d'ESO (~10-12 alumnes). **3 hores setmanals**: 2h al taller de bicicletes + 1h a l'aula maker (talladora làser, impressió 3D, càmera 360 i ulleres VR), ~35 setmanes ≈ **105 hores anuals**.

Metodologia ABP + ApS: es reparen bicicletes procedents de donacions que després es donen o queden a la comunitat educativa. El tercer trimestre inclou sortides urbanes setmanals en bici amb rutes virtualitzades en 360/VR, exposició pública final i **Certificat Intern de Mecànic/a Júnior de Bicicletes i Mobilitat Sostenible**.

Marc curricular: Decret 175/2022 (Catalunya), competències específiques de Tecnologia de 4t + competències transversals (C3, C5, C6, C8).

## Estructura del projecte

```
├── Programació didàctica\   Programació, temporització, mapatge competencial, diari setmanal
├── Classes\Bicicletes\B0…B9\  Unitats del taller de bicicletes
├── Classes\Maker\M0…M9\     Unitats de l'aula maker (blocs 3D → làser → 360)
├── Avaluació\               Rúbriques i instruments (fase 2)
├── Normativa\               Normes de taller, autoritzacions, protocol VR (fase 2)
├── Memòria de treball\      Diari docent, inventari, incidències (fase 2)
├── Recursos\                Material de referència (llibre de manteniment, fitxes, borrador previ)
└── docs\superpowers\        Spec i pla de treball
```
