# B6 — «Transmissió II: canvis i desviadors» (Taller de bicicletes)

| | |
|---|---|
| **Assignatura** | 🚲 Taller de bicicletes (2 h/setmana) |
| **Nivell** | 4t ESO · grup professionalitzador |
| **Setmanes** | 18–21 |
| **Producte** | Bici amb els **canvis ben ajustats** |

> 🔗 **Es complementa amb** l'hora de maker **M6** (caixa classificadora de recanvis), però són **dues assignatures separades**: aquí, només taller de bicicletes.

## 1. Descripció
Que una bici canviï de marxa suau i sense saltar separa una reparació correcta d'una excel·lent.

> **Repte:** «Ajusta els canvis perquè pugin i baixin net i sense saltar.»

## 2. Aprenentatge visible
En acabar seré capaç de…

- Ajustar topalls (H/L) i tensió del canvi posterior.
- Ajustar el canvi davanter i alinear-lo.
- Provar el canvi i corregir si salta o frega.

**Criteris d'èxit**

- [ ] He ajustat topalls i tensió del canvi posterior.
- [ ] He ajustat el canvi davanter.
- [ ] La bici canvia suau i no salta.

## 3. Sabers
- Canvi posterior i davanter; desviadors.
- Ajust: topalls H/L, tensor, alineació.
- Diagnòstic de problemes de canvi.

## 4. Sessions
**Setmana 18 — Canvi posterior: topalls i tensió**

1. Demo (25').
2. Ajust de topalls i tensió (65').
3. Diari (10').

**Setmana 19 — Canvi posterior: ajust fi**

1. Afinament amb el tensor i prova (90').
2. Diari (10').

**Setmana 20 — Canvi davanter i alineació**

1. Ajust del davanter i alineació (90').
2. Diari (10').

**Setmana 21 — Diagnosi encreuada**

1. Cada parella ajusta una bici i una altra la prova i comenta (100').

## 5. Atenció a la diversitat (DUA)
- Fitxa amb l'ordre d'ajust (primer topalls, després tensió).
- La diagnosi encreuada permet mostrar el que se sap ajudant.
- Qui domina ajuda els altres.
