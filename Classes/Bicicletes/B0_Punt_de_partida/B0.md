# B0 — «Punt de partida» (Taller de bicicletes)

| | |
|---|---|
| **Assignatura** | 🚲 Taller de bicicletes (2 h/setmana) |
| **Nivell** | 4t ESO · grup professionalitzador |
| **Setmanes** | 1–2 |
| **Producte** | **Carnet d'eines** + contracte d'aula + avaluació inicial |

> 🔗 **Es complementa amb** l'hora de maker **M0** (benvinguda i carnet de màquina), però són **dues assignatures separades**: aquí, només taller de bicicletes.

## 1. Descripció
Abans de tocar cap bici, cal conèixer el taller, les eines i treballar segur.

> **Repte:** «Mou-te pel taller amb seguretat i autonomia: guanya't el carnet d'eines.»

## 2. Aprenentatge visible
En acabar seré capaç de…
- Anomenar les eines bàsiques i la seva funció.
- Treballar seguint les normes de seguretat.
- Omplir la primera entrada del diari.

**Criteris d'èxit**
- [ ] He signat el contracte d'aula.
- [ ] Tinc el carnet d'eines.
- [ ] He fet l'avaluació inicial i la primera entrada del diari.

## 3. Sabers
- El taller: espai, zones, ordre i seguretat.
- Eines bàsiques: nom, funció, ús segur.
- Introducció a les parts de la bicicleta.

## 4. Sessions
**Setmana 1 — Benvinguda i seguretat**
1. Acollida i recorregut pel taller (25').
2. Les normes, raonades una a una (30').
3. Presentació de les eines amb la làmina de parts (35').
4. Tancament i diari (10').

**Setmana 2 — Gimcana i punt de partida**
1. Gimcana d'eines i seguretat (`Material_gimcana_taller.md`) (50').
2. Checkpoint del carnet d'eines (20').
3. Avaluació inicial pràctica (40').
4. Contracte + tancament (10').

## 5. Atenció a la diversitat (DUA)
- Demostracions físiques i làmina il·lustrada.
- Els checkpoints es superen fent (sense escriure).
- Èxit immediat el primer dia; parella d'ajuda.

## 6. Materials
- Eines bàsiques, bici de demostració, bomba i manòmetre.

- Fitxa: `Fitxa_alumnat.md` · Rúbrica: `Avaluació/Rubrica_bicicletes.md` · Diari: `Programació didàctica/Diari_setmanal_paper.pdf`
