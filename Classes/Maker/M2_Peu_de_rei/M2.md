# M2 — «Mesurar amb el peu de rei» (Aula maker)

| | |
|---|---|
| **Assignatura** | 🛠️ Aula maker (1 h/setmana) |
| **Nivell** | 4t ESO · grup professionalitzador |
| **Setmanes** | 6–8 |
| **Producte** | **Peça senzilla** dissenyada a mida real |
| **Bloc** | 🧊 3D |

> 🔗 **Es complementa amb** el taller de bicicletes (mesuraràs peces reials de la bici), però són **dues assignatures separades**: aquí, només aula maker.

## 1. Descripció
Per fabricar peces que encaixin cal mesurar bé. Aprens a fer servir el peu de rei i a passar les mides al disseny.

> **Repte:** «Mesura un objecte real amb el peu de rei i reprodueix-lo a mida a Tinkercad.»

## 2. Aprenentatge visible
En acabar seré capaç de…

- Llegir mides amb el peu de rei (en mm).
- Dibuixar un plànol 2D senzill acotat.
- Reproduir les mides a Tinkercad.

**Criteris d'èxit**

- [ ] Sé mesurar amb el peu de rei.
- [ ] He fet un plànol 2D amb les mides.
- [ ] He reproduït la peça a mida a Tinkercad.

## 3. Sabers
- El peu de rei: parts i lectura en mm.
- Cotes bàsiques i toleràncies (l'encaix).
- Del plànol 2D al model 3D.

## 4. Sessions
**Setmana 6 — El peu de rei**

1. Com es llegeix el peu de rei; pràctica amb objectes (30').
2. Plànol 2D acotat d'un objecte senzill (20').
3. Diari (10').

**Setmana 7 — Al Tinkercad**

1. Reproduir les mides al pla de treball (40').
2. Diari (10' + revisió de mides).

**Setmana 8 — Imprimir i comparar**

1. Imprimir i comparar amb l'original amb el peu de rei (40').
2. Diari (10').

## 5. Atenció a la diversitat (DUA)
- Peu de rei digital si costa la lectura de l'escala.
- Plantilla de plànol acotat.
- Es valida comparant amb l'original (visual).

## 6. Materials
- Peus de rei; objectes de mostra; Chromebooks; impressora 3D.
- Fitxa: `Fitxa_alumnat.md` · Rúbrica: `Avaluació/Rubrica_maker.md` · Diari: `Programació didàctica/Diari_setmanal_paper.pdf`
