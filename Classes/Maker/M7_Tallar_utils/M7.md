# M7 — «Tallar peces útils» (Aula maker)

| | |
|---|---|
| **Assignatura** | 🛠️ Aula maker (1 h/setmana) |
| **Nivell** | 4t ESO · grup professionalitzador |
| **Setmanes** | 22–24 |
| **Producte** | **Peça útil** tallada a làser per al taller |
| **Bloc** | ✂️ Làser |

> 🔗 **Es complementa amb** el taller de bicicletes (organitza i manté el taller), però són **dues assignatures separades**: aquí, només aula maker.

## 1. Descripció
Tallem peces útils per al taller: plaques, etiquetes, el regle de desgast de cadena i la caixa-kit.

> **Repte:** «Fabrica a làser una peça útil que el taller faci servir.»

## 2. Aprenentatge visible
En acabar seré capaç de…

- Triar una peça útil (placa, etiqueta, regle, caixa-kit).
- Preparar-ne el disseny amb els codis de color.
- Tallar-la i muntar-la.

**Criteris d'èxit**

- [ ] He triat i preparat una peça útil.
- [ ] L'he tallada correctament.
- [ ] Ja s'usa al taller.

## 3. Sabers
- Ús de les plantilles SVG (`Recursos/plantilles_svg/`).
- Caixes amb encaix amb MakerCase.
- Tall i muntatge.

## 4. Sessions
**Setmana 22 — Triar i preparar**

1. Triar la peça i obrir/adaptar la plantilla (40').
2. Diari (10').

**Setmana 23 — Tallar**

1. Tall a làser per tandes (40').
2. Diari (10').

**Setmana 24 — Muntar**

1. Muntatge i posada en ús al taller (40').
2. Diari (10').

## 5. Atenció a la diversitat (DUA)
- Plantilles ja fetes per començar (`Recursos/plantilles_svg/`).
- El que es fabrica queda a la vista al taller (reconeixement).
- Treball en parella.

## 6. Materials
- Chromebooks amb Inkscape; làser; MakerCase (per a les caixes); material de fabricació.
- Fitxa: `Fitxa_alumnat.md` · Rúbrica: `Avaluació/Rubrica_maker.md` · Diari: `Programació didàctica/Diari_setmanal_paper.pdf`
