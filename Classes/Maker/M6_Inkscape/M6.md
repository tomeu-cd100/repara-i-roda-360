# M6 — «Inkscape al Chromebook» (Aula maker)

| | |
|---|---|
| **Assignatura** | 🛠️ Aula maker (1 h/setmana) |
| **Nivell** | 4t ESO · grup professionalitzador |
| **Setmanes** | 18–21 |
| **Producte** | **Primer disseny 2D** preparat per a la làser |
| **Bloc** | ✂️ Làser |

> 🔗 **Es complementa amb** el taller de bicicletes (aviat tallaràs peces per al taller), però són **dues assignatures separades**: aquí, només aula maker.

## 1. Descripció
Comencem el bloc de làser: instal·lem Inkscape al Chromebook i aprenem a preparar un disseny per tallar.

> **Repte:** «Prepara el teu primer disseny 2D amb els codis de color correctes per a la làser.»

## 2. Aprenentatge visible
En acabar seré capaç de…

- Tenir Inkscape funcionant al Chromebook (o l'alternativa web).
- Dibuixar formes i text en mm.
- Aplicar els codis de color (tall / gravat / marca).

**Criteris d'èxit**

- [ ] Tinc Inkscape al Chromebook o he usat l'alternativa web.
- [ ] He fet un disseny senzill en mm.
- [ ] He aplicat els codis de color correctes.
- [ ] Tinc el carnet d'operador/a làser.

## 3. Sabers
- Instal·lació d'Inkscape al Chromebook (Linux/Crostini) o editor SVG web de reserva.
- Inkscape: formes, text, mides en mm.
- Codis de color de la làser: vermell = tall, negre = gravat, blau = marca.

## 4. Sessions
**Setmana 18 — Instal·lar**

1. Activar Linux (Crostini) i instal·lar Inkscape; alternativa web si cal (40').
2. Diari (10').

**Setmana 19 — Primers passos**

1. Interfície, formes, text i mides en mm (40').
2. Diari (10').

**Setmana 20 — Codis de color**

1. Aplicar vermell/negre/blau a un disseny (40').
2. Diari (10').

**Setmana 21 — Preparar per tallar**

1. Exportar/enviar a la làser; prova en material de mostra + checkpoint del carnet làser (40').
2. Diari (10').

## 5. Atenció a la diversitat (DUA)
- Guia il·lustrada d'instal·lació pas a pas.
- Alternativa web si Crostini no està disponible.
- Plantilles amb els colors ja definits (`Recursos/plantilles_svg/`).

## 6. Materials
- Chromebooks (Linux/Crostini) o editor SVG web; làser xTool S1; material de prova.
