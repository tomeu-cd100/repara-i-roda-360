# M0 — «Benvinguda maker i seguretat» (Aula maker)

| | |
|---|---|
| **Assignatura** | 🛠️ Aula maker (1 h/setmana) |
| **Nivell** | 4t ESO · grup professionalitzador |
| **Setmanes** | 1–2 |
| **Producte** | **Carnet de màquina** (làser i 3D) |
| **Bloc** | 🧊 3D |

> 🔗 **Es complementa amb** el taller de bicicletes (les peces que farem serviran per a les bicis), però són **dues assignatures separades**: aquí, només aula maker.

## 1. Descripció
L'aula maker fabrica peces i eines reals per al taller de bicicletes. Aquest curs treballem en tres blocs: primer impressió 3D, després tall làser i, si hi ha temps, un recorregut 360. Primer de tot: conèixer l'aula i treballar-hi amb seguretat.

> **Repte:** «Guanya't el carnet de màquina: demostra que saps operar la 3D i la làser amb seguretat.»

## 2. Aprenentatge visible
En acabar seré capaç de…

- Conèixer la impressora 3D i la làser i què fabricarem tot el curs.
- Saber les normes de seguretat de les màquines.

**Criteris d'èxit**

- [ ] Sé què fabricarem: peces 3D, després peces a làser i, si hi ha temps, el 360.
- [ ] Tinc el carnet de màquina (nivell làser i nivell 3D).

## 3. Sabers
- L'aula maker: impressora 3D Bambu Lab P2S i làser xTool S1.
- Els tres blocs del curs: 3D, làser i 360.
- Seguretat de màquines: materials, ventilació, parts calentes.

## 4. Sessions
**Setmana 1 — L'aula i el pla del curs**

1. Què fabricarem: exemples de peces per a la bici (20').
2. Les màquines i les normes de seguretat; demostració (30').
3. Diari (cara B) (10').

**Setmana 2 — Carnet de màquina**

1. Repàs de seguretat (15').
2. Checkpoint del carnet de màquina (`Normativa/Carnet_de_maquina.md`) (35').
3. Diari (10').

## 5. Atenció a la diversitat (DUA)
- Demostració física de les màquines.
- El checkpoint es supera fent (posar material, aturada d'emergència).
- Carnet 🔵 amb variant «guia» per a qui no operi.

## 6. Materials
- Impressora 3D i làser (demostració).
- Imprimible: `Normativa/Passaport_alumne.pdf`.
- Fitxa: `Fitxa_alumnat.md` · Rúbrica: `Avaluació/Rubrica_maker.md` · Diari: `Programació didàctica/Diari_setmanal_paper.pdf`
