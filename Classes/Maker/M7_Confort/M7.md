# M7 — «Peces de confort» (Aula maker)

| | |
|---|---|
| **Assignatura** | 🛠️ Aula maker (1 h/setmana) |
| **Nivell** | 4t ESO · grup professionalitzador |
| **Setmanes** | 22–24 |
| **Producte** | **Peces de confort i adaptadors** 3D, provats |

> 🔗 **Es complementa amb** l'hora de bicicletes **B7** (punts de contacte), però són **dues assignatures separades**: aquí, només aula maker.

## 1. Descripció
Fabriquem peces de confort i adaptadors a mida per a les bicis del taller.

> **Repte:** «Fabrica una peça de confort que millori una bici i prova-la.»

## 2. Aprenentatge visible
En acabar seré capaç de…

- Dissenyar peces de confort (topalls, proteccions, separadors).
- Imprimir adaptadors personalitzats.
- Provar i iterar les peces.

**Criteris d'èxit**

- [ ] He dissenyat una peça de confort 3D.
- [ ] L'he impresa i provada.
- [ ] L'he millorada si calia.

## 3. Sabers
- Modelatge 3D d'adaptadors a mida.
- Mesura de la persona/bici per personalitzar.

## 4. Sessions
**Setmana 22 — Disseny de peces de confort**

1. Disseny (topalls, proteccions, separadors) (50').
2. Diari (10').

**Setmana 23 — Adaptadors**

1. Impressió d'adaptadors personalitzats (50').
2. Diari (10').

**Setmana 24 — Prova i iteració**

1. Prova i iteració de les peces de confort (50').

## 5. Atenció a la diversitat (DUA)
- Referències de mida senzilles.
- Es prova fent (a la bici real).
- Personalització (motivador).

## 6. Materials
- Impressora 3D; material de fabricació.
- Fitxa: `Fitxa_alumnat.md` · Rúbrica: `Avaluació/Rubrica_maker.md` · Diari: `Programació didàctica/Diari_setmanal_paper.pdf`
