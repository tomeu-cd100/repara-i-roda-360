# M5 — «Mesurador de desgast» (Aula maker)

| | |
|---|---|
| **Assignatura** | 🛠️ Aula maker (1 h/setmana) |
| **Nivell** | 4t ESO · grup professionalitzador |
| **Setmanes** | 15–17 |
| **Producte** | **Mesurador de desgast de cadena** + clauer |

> 🔗 **Es complementa amb** l'hora de bicicletes **B5** (transmissió I), però són **dues assignatures separades**: aquí, només aula maker.

## 1. Descripció
Fabriquem un mesurador de desgast de cadena i un clauer per a la campanya de captació.

> **Repte:** «Fabrica un mesurador de desgast i prova'l a totes les cadenes del taller.»

## 2. Aprenentatge visible
En acabar seré capaç de…
- Dissenyar una eina de mesura amb mides crítiques.
- Fabricar-la amb làser o 3D.
- Provar-la a cadenes reals.

**Criteris d'èxit**
- [ ] He dissenyat el mesurador amb la mida crítica.
- [ ] L'he fabricat.
- [ ] Funciona a les cadenes del taller.

## 3. Sabers
- Fabricació d'eines de mesura amb toleràncies.
- Disseny 2D/3D d'objectes petits i precisos.

## 4. Sessions
**Setmana 15 — Disseny del mesurador**
1. Disseny amb la mida crítica (50').
2. Diari (10').

**Setmana 16 — Fabricació + clauer**
1. Fabricació del mesurador + clauer-engranatge (50').
2. Diari (10').

**Setmana 17 — Prova i ajust**
1. Prova a les cadenes del taller; ajust (50').

## 5. Atenció a la diversitat (DUA)
- El desgast es «llegeix» amb el mesurador (visual).
- Plantilla de mides.
- El mesurador s'usa de seguida (utilitat).

## 6. Materials
- Làser i/o impressora 3D; material de fabricació.
- Fitxa: `Fitxa_alumnat.md` · Rúbrica: `Avaluació/Rubrica_maker.md` · Diari: `Programació didàctica/Diari_setmanal_paper.pdf`
