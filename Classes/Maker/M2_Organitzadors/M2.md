# M2 — «Organitzadors d'eines» (Aula maker)

| | |
|---|---|
| **Assignatura** | 🛠️ Aula maker (1 h/setmana) |
| **Nivell** | 4t ESO · grup professionalitzador |
| **Setmanes** | 5–7 |
| **Producte** | **Organitzadors i penjadors d'eines** + retolació del taller |

> 🔗 **Es complementa amb** l'hora de bicicletes **B2** (posada a punt i M-check), però són **dues assignatures separades**: aquí, només aula maker.

## 1. Descripció
Fabriquem el que el taller necessita per treballar ordenats: organitzadors i retolació.

> **Repte:** «Dissenya i fabrica un organitzador d'eines útil per al taller.»

## 2. Aprenentatge visible
En acabar seré capaç de…

- Mesurar les eines reals abans de dissenyar-ne el suport.
- Dissenyar i fabricar amb làser o 3D.
- Retolar les zones del taller.

**Criteris d'èxit**

- [ ] He mesurat les eines i dissenyat l'organitzador.
- [ ] He fabricat l'organitzador i ja s'usa al taller.
- [ ] He col·laborat en la retolació.

## 3. Sabers
- Disseny per fabricació: mesurar l'objecte real.
- Tall làser i/o impressió 3D d'objectes funcionals.

## 4. Sessions
**Setmana 5 — Disseny**

1. Mesurar les eines i dissenyar l'organitzador (50').
2. Diari (10').

**Setmana 6 — Fabricació**

1. Tall làser o impressió 3D de l'organitzador (50').
2. Diari (10').

**Setmana 7 — Muntatge i retolació**

1. Muntatge i retolació de les zones del taller (50').
2. Diari (10').

## 5. Atenció a la diversitat (DUA)
- Versió base senzilla de l'organitzador.
- El que es fabrica queda a la vista al taller (reconeixement).
- Treball per parelles.

## 6. Materials
- Ordinadors, làser i/o impressora 3D; material de fabricació.
- Fitxa: `Fitxa_alumnat.md` · Rúbrica: `Avaluació/Rubrica_maker.md` · Diari: `Programació didàctica/Diari_setmanal_paper.pdf`
