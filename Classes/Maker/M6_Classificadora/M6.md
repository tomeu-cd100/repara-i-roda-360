# M6 — «Caixa classificadora» (Aula maker)

| | |
|---|---|
| **Assignatura** | 🛠️ Aula maker (1 h/setmana) |
| **Nivell** | 4t ESO · grup professionalitzador |
| **Setmanes** | 18–21 |
| **Producte** | **Caixa classificadora de recanvis** + magatzem ordenat |

> 🔗 **Es complementa amb** l'hora de bicicletes **B6** (transmissió II), però són **dues assignatures separades**: aquí, només aula maker.

## 1. Descripció
Organitzem el magatzem de recanvis amb una caixa classificadora modular.

> **Repte:** «Fabrica una caixa classificadora i ordena el magatzem de recanvis.»

## 2. Aprenentatge visible
En acabar seré capaç de…

- Dissenyar una caixa modular amb compartiments.
- Tallar i muntar la classificadora.
- Organitzar i retolar el magatzem.

**Criteris d'èxit**

- [ ] He dissenyat la classificadora (mòduls).
- [ ] L'he tallada i muntada.
- [ ] El magatzem de recanvis està ordenat.

## 3. Sabers
- Disseny modular per a làser (encaixos).
- Organització i retolació d'un magatzem.

## 4. Sessions
**Setmana 18 — Disseny**

1. Disseny de la caixa classificadora (mòduls) (50').
2. Diari (10').

**Setmana 19 — Fabricació**

1. Tall làser i muntatge (50').
2. Diari (10').

**Setmana 20 — Suports**

1. Suports i millores per als peus de treball (50').
2. Diari (10').

**Setmana 21 — Organització final**

1. Organització del magatzem amb la classificadora (50').

## 5. Atenció a la diversitat (DUA)
- Plantilla modular.
- El resultat s'usa al magatzem (reconeixement).
- Treball per parelles.

- Làser; material de fabricació.
