# M5 — «Peces decoratives temàtica bici (3D)» (Aula maker)

| | |
|---|---|
| **Assignatura** | 🛠️ Aula maker (1 h/setmana) |
| **Nivell** | 4t ESO · grup professionalitzador |
| **Setmanes** | 15–17 |
| **Producte** | **Objecte decoratiu** propi (temàtica bici) |
| **Bloc** | 🧊 3D |

> 🔗 **Es complementa amb** el taller de bicicletes (temàtica ciclista; per a l'exposició i els certificats), però són **dues assignatures separades**: aquí, només aula maker.

## 1. Descripció
Dissenyem objectes decoratius amb temàtica de bici per a l'exposició i els records del curs.

> **Repte:** «Dissenya i imprimeix un objecte decoratiu amb temàtica de bici.»

## 2. Aprenentatge visible
En acabar seré capaç de…

- Triar un objecte decoratiu (clauer baula, tapa de vàlvula, imant silueta, pin «Mecànic Júnior», marcador).
- Dissenyar-lo a Tinkercad.
- Imprimir-lo i acabar-lo.

**Criteris d'èxit**

- [ ] He triat i dissenyat un objecte decoratiu.
- [ ] L'he imprès.
- [ ] Té bon acabat per a l'exposició.

## 3. Sabers
- Disseny 3D creatiu (estètica).
- Text i relleu a Tinkercad.
- Acabats.

## 4. Sessions
**Setmana 15 — Idea i disseny**

1. Triar l'objecte i dissenyar-lo (text/relleu) (40').
2. Diari (10').

**Setmana 16 — Impressió**

1. Impressió per tandes (40').
2. Diari (10').

**Setmana 17 — Acabats**

1. Neteja, poliment i preparació per a l'exposició (40').
2. Diari (10').

## 5. Atenció a la diversitat (DUA)
- Galeria d'exemples per inspirar-se.
- Plantilla amb el text ja posat.
- Producte personal (motivador).

## 6. Materials
- Chromebooks; impressora 3D; material d'acabats.
- Fitxa: `Fitxa_alumnat.md` · Rúbrica: `Avaluació/Rubrica_maker.md` · Diari: `Programació didàctica/Diari_setmanal_paper.pdf`
